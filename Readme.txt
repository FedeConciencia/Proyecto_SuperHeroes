Proyecto FullStak SuperHeroes (Cliente-Servidor) aplicando tecnologias:

FrontEnd => ReactJs, JavaScript
BackEnd => Java, Servlets
BaseDatos => MySql (relacional)

En el cliente se puede acceder a una pagina web con diferentes accesos donde se puede
visualizar los SuperHeroes sus caracteristicas, cuenta con una seccion de administrador
donde se accede al CRUD- AMB de las entidades que componen el proyecto. Las peticiones son
gestionadas por el cliente al servidor java a traves de las clases servlet, devolviendo
la informacion solicitada al cliente.
